# Harbor of Echoes Image Prompt Pack

These prompts are written for project-bound asset generation.

## Asset 01: Homepage Hero Background

Filename target:

- `public/images/hero-harbor-night.png`

Prompt:

```text
Use case: stylized-concept
Asset type: landing page hero background
Primary request: a cinematic coastal harbor at night for a narrative investigation webapp
Scene/backdrop: storm-beaten small harbor with wet concrete, low industrial lights, distant tidal infrastructure, mist over black water, no people visible
Subject: the environment itself, with emphasis on secrecy and tension
Style/medium: high-end editorial concept art with realistic textures
Composition/framing: wide horizontal composition with substantial negative space for interface copy, horizon kept low, strongest light sources clustered far right
Lighting/mood: cold moonless night, sodium lamps, damp haze, restrained contrast, ominous and quiet
Color palette: abyss blues, slate greys, muted amber lights, salt-paper highlights
Materials/textures: wet asphalt, corroded metal, harbor ropes, concrete barriers, sea spray
Constraints: no text, no logos, no boats in the foreground, no cinematic orange-teal exaggeration, no watermark
Avoid: cyberpunk lighting, fantasy architecture, crowded composition
```

## Usage Notes

- Keep all images text-free unless the final UI will add the text later.
- Prefer atmospheric realism over stylization when the asset is framed as evidence.
- Use non-destructive versioning if generating multiple candidates, for example `hero-harbor-night-v2.png`.
